ERROR CODE : -14

Command pos. has suddenly changed

Remarks
동작 지령치가 모터의 허용 가능 최고 속도보다 클 때 발생.
Repeat 모드에서는 최대 모터 속도(기본값은 정격 속도의 2배)를 초과하여 joint 이동 명령이
지시되었을 때 발생. Check 모드에서는 250mm/sec의 속도를 초과하여
joint 이동 명령이 지시 되었을 때도 발생

=> 로봇 config 에서 MAX_RPM으로 설정된 모터의 최대 속도 설정이 올바른지 확인한다.
정의되어 있지 않을 시 RPM의 2배로 설정된다. => 로봇 모션을 하는 동안 singular 조건을 확인하고, singular 위치 근처를 벗어난 위치로 teach한다.
=> 표준 6축 수직다관절형 로봇일 경우, 손목축의 Singular 점 통과시 MoveH(6축 옵션) 의 동작 명령을 사용해 본다
=> Singular 위치 근처를 통과시 속도를 허용속도 이내로 줄이기 바랍니다.
=> Singular 위치 근처를 통과시 명령어를 Joint move 명령을 사용해 본다.
=> 컨베이어 트랙 명령등을 사용시, 트랙킹 간 이동시 발생할 수 있음.
이 경우엔, Break 명령을 통해, 모션을 완료 후 다른 모션으로 진행하도록 합니다.

ERROR CODE : -15

Servo driver pos. and command has too much difference

Remarks
실시간 동작 지령치 이상을 감시하는 옵션을 사용할 경우에만 발생할 수 있으며
제어기의 위치 지령치가 모터의 허용 가능 최고 속도보다 클 때 발생.
모터의 허용 가능 최고 속도는 로봇 설정에서 MAX_RPM으로 설정되고, 설정되어 있지 않을 시 기본값으로 정격속도(RPM)의 2배로 설정된다
=> 모터의 최대 속도 설정이 올바른지 확인
=> 로봇 기구의 Singular 자세 주변에서의 동작명령인지 확인
