int CUIEcat::getDecErrorState ( unsigned int &  value)

Dec Error State, 현재 Master Error 상태

Parameters
[out]value
현재 Master Error 상태 변수
bit 0: 현재 Slave 개수 Error 상태, 참조: CUIEcat::getOnlineSlaveNumErrorCount()
bit 1: Ready Event Error 상태, 참조: CUIEcat::getRdyEventErrorCount()
bit 2: App Event Error 상태, 참조: CUIEcat::getAppEventErrorCount()
bit 3: End Event Error 상태, 참조: CUIEcat::getEndEventErrorCount()
bit 4: Wait Period Error 상태, 참조: CUIEcat::getWaitPeriodErrorCount()
bit 5: Deep Sleep Error 상태, 참조: CUIEcat::getDeepSleepErrorCount()
bit 6: Wakeup Count Error 상태, 참조: CUIEcat::getWakeupLargeErrorCount()
bit 7: 현재 Error 상태 Flag
Returns
success 0
